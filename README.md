# SavannahShop

Django e-commerce application with Keycloak OIDC authentication and Celery for async tasks.

## Features

- Django 4.2+
- Keycloak OIDC (mozilla-django-oidc)
- Celery with Redis
- PostgreSQL database
- Hot-reload development
- Kubernetes deployment ready

## Development

### Quick Start with Hot Reload

```bash
./scripts/dev.sh
```
